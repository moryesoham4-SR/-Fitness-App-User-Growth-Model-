#  Fitness App Growth Model

##  Overview
This project predicts the growth of users in a fitness app using mathematical modeling.

## 🎯Objectives
- Predict user growth over time
- Analyze retention and churn
- Visualize multiple growth models

##  Models Used
- Exponential Growth
- Logistic Growth
- Retention Model

##  Features
- Interactive Streamlit dashboard
- User input parameters
- Multiple graph outputs

##  Dataset
Workout dataset used to simulate user behavior.
